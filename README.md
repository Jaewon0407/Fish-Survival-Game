## Installation

1. Install SDL2
2. Install SDL2_Image
3. Install SDL2_TTF

## Game Description

Interactive 2D survival game where the primary object is to grow as large as possible by consuming smaller fish while avoiding getting eaten by bigger fish.

## Controls 

UP,
RIGHT,
DOWN,
LEFT
